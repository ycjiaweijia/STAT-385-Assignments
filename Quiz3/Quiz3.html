Question 5 [2pt]

Load the R package MASS and use the data frame cats. Obtain vectors sex and bwt from the data frame with sex <- cats$Sex, bwt <- cats$Bwt. Show the body weights of female cats by creating vector bwtF. How many female cats are there in the data set? Please save the length value in object len.



library(MASS)
sex <- cats$Sex
bwt <- cats$Bwt
bwtF<- bwt[sex=="F"]
bwtF
#>  [1] 2.0 2.0 2.0 2.1 2.1 2.1 2.1 2.1 2.1 2.1 2.1 2.1 2.2 2.2 2.2 2.2 2.2 2.2 2.3
#> [20] 2.3 2.3 2.3 2.3 2.3 2.3 2.3 2.3 2.3 2.3 2.3 2.4 2.4 2.4 2.4 2.5 2.5 2.6 2.6
#> [39] 2.6 2.7 2.7 2.7 2.9 2.9 2.9 3.0 3.0
len <- length(bwtF)
len <- sum(sex=="F")
